# BOJ
## 1. submit.rb 사용법
<code>ruby submit.rb</code> 로 실행합니다.
코드의 요구사항은 다음과 같습니다.
- github repository : 처음 설정 후 다시 묻지 않습니다.
- user.email : 코드 내부에서 변경이 가능합니다.
- user.name : 코드 내부에서 변경이 가능합니다.
- (c++ source) path : 항상 물어봅니다. (상대경로, 절대경로 둘 다 가능합니다.)
